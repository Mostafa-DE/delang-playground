# Delang Playground

Welcome to the Delang Playground, an interactive web-based environment for writing, running, and experimenting with the DE programming language.

![DE Logo](https://res.cloudinary.com/dcj3o6yi5/image/upload/v1694814823/delang-rocks_lvj0zm.png)

## Getting Started

To get started with the Delang Playground, visit the website at [https://delang.mostafade.com](https://delang.mostafade.com).

## Features

- **Interactive Environment**: Write DE code and see the results instantly.
- **Syntax Highlighting**: Code is displayed with syntax highlighting for improved readability.
- **History**: Access your code history to revisit and edit previous sessions (Coming Soon).
- **Share Your Code**: Easily share your DE code snippets with others (Coming Soon).

## Usage

Using the Delang Playground is straightforward:

1. Visit [https://delang.mostafade.com](https://delang.mostafade.com).
2. Read the explanation on the left.
3. In the editor on the right, write your DE code.
4. Click the "Run" button to execute your code.
5. View the results in the console on the bottom right.

## Examples

Here are some sample DE code snippets to help you get started:

  ```de
  logs("Hello, World!");
- **Calculate the Sum**
  ```de
  const a = 5;

- **Define a Function**
  ```de

  };
